Installation
============

The following command line binaries are required:
* fping

The following Perl packages are required:
* JSON(::XS)
* Mojolicious
* SNMP
* Statistics::Basic
* Text::Template

You need to checkout the following git submodules to get
a working copy of Scotty RELOADED:
* qTip2
* spin.js
* web-socket-js
